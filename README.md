###1.1 함수의 호출 방식

함수의 인자를 전달하는 방식
* 값에 의한 호출 (Call by Value)

    * 함수에 인자를 넘길 때 값만 넘김.
    * 함수내에 인자값 변경시 호출자에 영향을 주지 않

* 참조에 의한 호출 (Call by Reference)음
    *함수에 인자를 넘길 때 메모리 주소를 넘김
    *함수내에 인자값 변경시, 호출자의 값도 변겨오딤
